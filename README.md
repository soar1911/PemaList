# 牌位資料處理系統

這是一個用於處理牌位資料的 Web 應用程式，可以將 Excel 檔案轉換為特定格式的 Word 文件。

## 功能

- 上傳 Excel 檔案
- 生成消災牌位文件
- 生成超薦牌位文件
- 生成功德主名單
- 生成報到名單
